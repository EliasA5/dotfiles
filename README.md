my tmux configuration

Installation:
1. clone the repository:
    ```bash
    git clone https://github.com/EliasA5/tmux.conf ~/.config/tmux
    ```
2. install tpm:
    ```bash
    git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
    ```
2. open tmux, and hit `<C-Space>` then `I`.
3. Enjoy.
